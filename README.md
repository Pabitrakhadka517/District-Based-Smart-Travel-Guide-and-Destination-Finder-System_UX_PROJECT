# District-Based Smart Travel Guide and Destination Finder System

NepaYatra is a full-stack travel guide platform for exploring Nepal's 77 districts — destinations, treks, festivals, guides, bookings, and an interactive map — with a Node/Express/MongoDB backend and a Next.js frontend.

## Structure

- `Backend/` — REST API (Express, MongoDB/Mongoose, JWT auth, Cloudinary, Swagger docs)
- `Frontend/` — Next.js client app
- `docs/` — supplementary documentation
